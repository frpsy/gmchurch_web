# gmchurch_web

gmchurch_web은 웹 기반의 GMChurch 관리 도구입니다. 이 프로젝트는 교회내 다양한 활동을 지원하고, 신도 관리, 일정 관리 및 공지사항 추가 등 여러 기능을 제공합니다.

## 설치 및 실행 방법

### 1. 코드 클론하기
```bash
git clone https://github.com/frpsy/gmchurch_web.git
cd gmchurch_web
```

### 2. 의존성 설치
사용하는 패키지 관리자를 통해 의존성을 설치합니다.
```bash
npm install
# 또는
yarn install
```

### 3. 개발 서버 실행
```bash
npm start
# 또는
yarn start
```

## 기술 스택
- **프로그래밍 언어:** JavaScript, HTML, CSS
- **프레임워크:** React, Node.js
- **기타 라이브러리:** Express, Axios, etc.

1. 이 저장소를 `fork`합니다.
2. 새로운 브랜치를 생성합니다. (`git checkout -b feature/AmazingFeature`)
3. 변경 사항을 커밋합니다. (`git commit -m 'Add some AmazingFeature'`)
4. 원본 브랜치로 `push`합니다. (`git push origin feature/AmazingFeature`)
5. Pull Request를 생성합니다.
